Posíláme data ze stránky na server

Pro odesílání jakýchkoliv dat na server se využívají formuláře, které obsahují vstupní pole - nepárové elementy <input>. ty mohou mít více vlastností, atributů, co ovlivňují jejich chování.

<form action="stranka.php" method="POST">
    <input type="text" name="jmeno" value="hodnota">
    <input type="button" value="Text tlačítka">
</form>

Základní typy inputů

Textové vstupní pole

type="text"

  • placeholder="" - popis, ukázkový text...
  • maxlength="" - max. délka textu

Číselné vstupní pole

type="number"

  • min="" - minimální hodnota
  • max="" - maximální hodnota
  • step="" - krokování pomocí šipek

Vstupní pole pro heslo

type="password"

  • placeholder="" - popis, ukázkový text...
  • maxlength="" - max. délka textu

Vstupní pole pro soubor

type="file"
Pokročilejší pracování na úrovni PHP.

Zaškrtávací pole

type="checkbox"
Doplňuje se o tag label.

Výběrové pole

type="radio"
Doplňuje se o tag label.

Tlačítko

type="submit"
